PixelTrail 组件使用说明

1. 基本用法

在HTML中直接使用组件标签：

<pixel-trail 
  :grid-size="50" 
  :trail-size="0.15" 
  color="#00ff00">
</pixel-trail>

2. 可用属性

  • gridSize (Number): 网格大小，默认为50
  • trailSize (Number): 轨迹大小，默认为0.15
  • maxAge (Number): 最大年龄值，控制轨迹持续时间，默认为300
  • interpolate (Number): 插值参数，控制平滑度，默认为8
  • color (String): 轨迹颜色，默认为'#00ff00'
  • className (String): 自定义类名
  • gooeyFilter (Object): 润滑效果参数，包含id和strength属性

3. 示例代码

<pixel-trail 
  :grid-size="40"
  :trail-size="0.2"
  :max-age="200"
  :interpolate="10"
  color="#ff00ff"
  class-name="custom-pixel-trail">
</pixel-trail>

4. 注意事项

此组件依赖 THREE.js 库，请确保在使用前已正确引入该库。像素轨迹效果适用于鼠标移动追踪等交互场景。
