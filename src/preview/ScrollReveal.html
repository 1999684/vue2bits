ScrollReveal 组件使用说明

1. 基本用法

在HTML中直接使用组件标签：

<scroll-reveal :children="'显示文本'"></scroll-reveal>

2. 可用属性

  • children (String): 要显示的文本内容
  • scroll-container-ref (Object): 滚动容器引用，默认为window
  • enable-blur (Boolean): 是否启用模糊效果，默认为true
  • base-opacity (Number): 基础透明度，默认为0.1
  • base-rotation (Number): 基础旋转角度，默认为3
  • blur-strength (Number): 模糊强度，默认为4
  • container-class-name (String): 应用于容器的CSS类名
  • text-class-name (String): 应用于文本的CSS类名
  • rotation-end (String): 旋转动画结束位置，默认为'bottom bottom'
  • word-animation-end (String): 文字动画结束位置，默认为'bottom bottom'

3. 示例代码

<scroll-reveal 
    :children="message" 
    text-class-name="text-try"
    container-class-name="text-center"
    :enable-blur="true"
    :base-opacity="0.1"
    :base-rotation="3"
    :blur-strength="4"
    :rotation-end="'bottom bottom'"
    :word-animation-end="'bottom bottom'"
>
</scroll-reveal>

4. 自定义样式

可以通过text-class-name属性添加自定义CSS类来改变文本样式：

.text-try {
  color: #000;
}
