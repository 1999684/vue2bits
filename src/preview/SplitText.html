SplitText 组件使用说明

1. 基本用法

在HTML中直接使用组件标签：

<split-text :text="'显示文本'"></split-text>

2. 可用属性

  • text (String): 要显示的文本内容
  • class-name (String): 应用于文本的CSS类名
  • duration (Number): 动画持续时间（秒），默认为0.6
  • delay (Number): 字符动画的间隔时间（毫秒），默认为100
  • ease (String|Function): 动画缓动函数，默认为'power3.out'
  • split-type (String): 分割类型('chars'|'words'|'lines')，默认为'chars'
  • text-align (String): 文本对齐方式，默认为'center'
  • from (Object): 动画起始状态，默认为{ opacity: 0, y: 40 }
  • to (Object): 动画结束状态，默认为{ opacity: 1, y: 0 }

3. 示例代码

<split-text 
    :text="message" 
    :duration="3"
    :delay="200"
    text-align="center"
    class-name="demo-text"
    :from="{
        opacity: 0.1,
        y: 20
    }"
    :to="{
        opacity: 1,
        y: 0
    }"
    :threshold="0.1"
    :on-letter-animation-complete="toshow"
>
</split-text>
