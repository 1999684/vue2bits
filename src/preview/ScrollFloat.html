ScrollFloat 组件使用说明

1. 基本用法

在HTML中直接使用组件标签：

<scroll-float :children="'显示文本'"></scroll-float>

2. 可用属性

  • children (String): 要显示的文本内容
  • animation-duration (Number): 动画持续时间（秒），默认为1
  • text-class-name (String): 应用于文本的CSS类名
  • container-class-name (String): 应用于容器的CSS类名
  • ease (String): 动画缓动函数，默认为'back.inOut(2)'
  • scroll-start (String): 滚动触发动画的起始位置
  • scroll-end (String): 滚动触发动画的结束位置
  • stagger (Number): 字符动画的间隔时间，默认为0.03

3. 示例代码

<scroll-float
    :children="message"
    :animation-duration="0.1"
    text-class-name="text-try"
    container-class-name="text-center"
    :ease="'back.inOut(2)'"
    :scroll-start="'bottom 95%'"
    :scroll-end="'top 50%'"
    :stagger="0.5"
>
</scroll-float>

4. 自定义样式

可以通过text-class-name属性添加自定义CSS类来改变文本样式：

.text-try {
  color: #000;
}
