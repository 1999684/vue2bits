TargetCursor 组件使用说明

1. 基本用法

在 HTML 中放置组件，它会自动接管鼠标指针：

<target-cursor target-selector=".cursor-target"></target-cursor>

2. 可用属性

  • target-selector (String): 触发锁定效果的 CSS 选择器，默认为 '.cursor-target'
  • spin-duration (Number): 闲置时光标旋转一周的秒数，默认为 2
  • hide-default-cursor (Boolean): 是否隐藏系统默认光标，默认为 true
  • color (String): 光标颜色（支持 CSS 颜色值），默认为 'white'

3. 视觉原理

当鼠标进入匹配 target-selector 的元素时，光标的四个角会平滑地展开并贴合到目标的边界，同时停止旋转并产生轻微的视差跟随效果。

4. 依赖说明

本组件依赖于 GSAP。请确保在预览环境或全局已加载 GSAP 库：

<script src="../libs/gsap.min.js"></script>
